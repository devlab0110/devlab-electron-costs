# devlab-electron-costs
Monthly cost editor based on electron

### Setup
```
node v18
npm install
```

### Test
```
npm start
```

### Buld .deb
```
npm run make
```
find .deb install file in **./out/make/deb/x64/monthly-cost_1.0.0_amd64.deb**
and install this deb pakage / this app
